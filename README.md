# Python Programming Language

Python is a popular, beginner-friendly programming language used for web development, automation, data analysis, artificial intelligence, machine learning, scripting, and many other tasks.

## Why Learn Python?

- Simple and readable syntax
- Large community and strong documentation
- Useful for beginners and professionals
- Supports many fields, including web apps, AI, data science, and automation
- Huge collection of libraries and frameworks

## Basic Example

```python
print("Hello, Python!")
```

Output:

```text
Hello, Python!
```

## Variables

```python
name = "Karan"
age = 21

print(name)
print(age)
```

## Strings

```python
message = "Python is easy to learn"

print(message)
print(message.upper())
print(message.lower())
```

## Lists

```python
languages = ["Python", "Java", "C++"]

print(languages[0])
languages.append("JavaScript")
print(languages)
```

## Functions

```python
def greet(name):
    return "Hello, " + name

print(greet("Karan"))
```

## Common Uses of Python

- Web development with Django or Flask
- Data analysis with pandas
- Machine learning with scikit-learn or TensorFlow
- Automation scripts
- Game development
- Desktop applications
- API development

## How to Run Python

1. Install Python from https://www.python.org/
2. Create a file named `main.py`
3. Write Python code inside the file
4. Run it using:

```bash
python main.py
```

On some systems, use:

```bash
python3 main.py
```
