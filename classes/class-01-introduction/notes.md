# Class 01: Introduction to Python

## What is Python?

Python is a programming language used to give instructions to a computer. It is popular because the code is easy to read and write.

## Topics Covered

- What programming is
- What Python is used for
- How to print output
- What variables are
- Basic data types

## Print Statement

The `print()` function is used to show output on the screen.

```python
print("Hello, Python!")
```

## Variables

A variable stores a value.

```python
name = "Karan"
age = 21
```

## Basic Data Types

- `str`: text, like `"Python"`
- `int`: whole numbers, like `10`
- `float`: decimal numbers, like `10.5`
- `bool`: true or false values, like `True`
