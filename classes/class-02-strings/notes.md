# Class 02: Strings and Text

## What is a String?

A string is text written inside quotes.

```python
message = "Python is easy"
```

## Multi-line Strings

Use triple quotes for text with multiple lines.

```python
paragraph = """This is my new car.
My car is very fast.
My car has white colour."""
```

## Count Text

The `.count()` method counts how many times a word or letter appears.

```python
paragraph.count("car")
```

## Replace Text

The `.replace()` method replaces one word with another.

```python
paragraph.replace("My", "Our")
```
